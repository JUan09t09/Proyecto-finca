⚙️ Configuración inicial

Para sincronizar tu finca entre dispositivos, necesitas conectar Firebase (gratuito).

1 Ve a console.firebase.google.com e inicia sesión con tu cuenta Google.
2 Crea un proyecto nuevo → pon cualquier nombre (ej: mi-finca) → desactiva Google Analytics → Crear proyecto.
3 En el proyecto, haz clic en el ícono </> (Web) → registra la app → copia la configuración firebaseConfig.
4 En el menú izquierdo: Authentication → Comenzar → Email/Contraseña → Habilitar.
5 En el menú izquierdo: Firestore Database → Crear base de datos → Modo de prueba → Listo.
🔒 Privacidad: Tu configuración se guarda solo en este navegador. Firebase es gratuito para uso personal (hasta 1GB de datos, más que suficiente para una finca).
